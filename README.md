# Lexiuridicus

Sitio web corporativo y panel administrativo para promoción de servicios jurídicos empresariales, captación de clientes potenciales, gestión de solicitudes, administración de blog y moderación de comentarios.

## Stack técnico

- Next.js 15
- React 19
- TypeScript
- MySQL 8
- ESLint 9
- Zod
- GitHub Actions

## Requisitos

- Node.js 22 recomendado
- MySQL 8 o compatible
- npm 10 o superior

## Inicio rápido

### 1. Instalar dependencias

```bash
npm install
```
